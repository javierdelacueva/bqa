Welcome to Backbonejs Quick App (BQA 1.0)

BQA 1.0 is a simple and light framework to help you building Apps using Backbonejs and Parse.com. Taken advantage of the MVC pattern that backbonejs brings to the client side and the awesome ease of Parse.com to save data and interact with its API, we have build a simple but useful framework that will allow you to build an Application in less than 5 minutes.

This is the first version of this framework which includes login/signup functionality and a basic CRUD with pagination for each class that you want to create.

Please take a look into the help section to learn how to start working with BQA and remember that your feedback is really appreciated!
